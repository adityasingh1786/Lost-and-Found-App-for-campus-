# 🏫 Campus Lost & Found Application
"A Flask-based web platform for university students to report, track, and claim misplaced items on campus."

A centralized platform designed for university students to report lost items and browse found belongings. This project was developed as part of a **Project Based Learning (PBL)** initiative at Sharda University to solve the common issue of misplaced items on a large campus.

## 🌟 Key Features
* **Report & Log:** Easily submit details about a lost or found item, including category and location.
* **Live Feed:** A real-time dashboard to browse all reported items.
* **Personal Dashboard:** A "My Reports" section where users can track the status of their own submissions.
* **Search & Filter:** Quickly find specific items based on keywords or categories.
* **Claiming System:** A workflow for users to verify and claim found items.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Frontend:** HTML5, CSS3, JavaScript
* **Data Management:** Pandas & CSV (designed for lightweight, efficient local storage)
* **Logic:** Custom JavaScript for dynamic UI updates

## 📂 Project Structure
* `LOSTAPP.py`: Core Flask application handling routes and data logic.
* `index1.html`: The main user interface for reporting and browsing.
* `script.js`: Handles frontend validation and dynamic interactions.
* `data.csv`: Local database for storing item records.
